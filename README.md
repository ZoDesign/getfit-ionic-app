# getfit-ionic-app
GetFit – Ionic Fitness App

GetFit is a mobile application developed using Ionic, Angular, and TypeScript for a small health and wellness 
business that provides personalised fitness plans. The app allows users to sign up or log in, browse workout 
programs, view workout details, and track their progress. All data is stored locally using local storage.

------------------------------------------------------------
## 1. FEATURES
------------------------------------------------------------

### ✓ User Authentication
- Users can sign up and log in.
- Authentication is simulated using local storage.
- Login and signup forms use Ionic Input components.

### ✓ Workout Program Listing
- Users can browse a list of predefined workout categories 
  (Weight Loss, Muscle Gain, Cardio, etc.).
- Programs are managed using an Angular Workout Service.
- Workout programs displayed using Ionic List components.

### ✓ Workout Details
- When a workout is selected, the user navigates to a 
  dedicated details page.
- Details include:
  • Workout description  
  • List of exercises  
  • Duration and equipment needed  
- Users can mark workouts as completed.

### ✓ Progress Tracking
- Completed workouts are saved using Ionic Storage.
- A Progress page displays the user’s achievements.
- Users have an option to reset their progress.

### ✓ Navigation & UI
- App includes the following pages:
  • Home  
  • Workouts  
  • Workout Details  
  • Progress  
  • Login  
  • Sign Up  
- Navigation implemented using Ionic Router.
- UI built using Ionic components with custom SCSS styling.

## 3. HOW TO RUN THE PROJECT
------------------------------------------------------------

Step 1: Install Ionic CLI
Run:
    npm install -g @ionic/cli


Step 2: Run the Application
Use:
    ionic serve
